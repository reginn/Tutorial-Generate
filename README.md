Tutorial-Generate
=================

Minecraft 1.6.x Modding Tutorial "Generate" with Minecraft Forge #943.

実行する際はcomフォルダをsrc/minecraft/に, assetsフォルダをbin/minecraft/にコピーしてください.

Eclipse上で実行する場合, assetsフォルダはeclipse/minecraft/bin/assets/に上書きしてください.
